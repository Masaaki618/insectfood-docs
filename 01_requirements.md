# 01_requirements.md｜昆虫食初心者ガイド 要件定義書

---

## 1. アプリ概要

| 項目    | 内容                                       |
|-------|------------------------------------------|
| 目的    | 昆虫食に興味があるが未経験の初心者が、自分に合った昆虫食を見つけられるようにする |
| ターゲット | 昆虫食未経験の初心者（最初の一歩を踏み出したい人）                |
| 規模    | ユーザー数 〜100人想定                            |

---

## 2. MVP機能

| 機能     | 概要                                                     |
|--------|--------------------------------------------------------|
| 昆虫食診断  | カテゴリ別にランダムで6問出題し、虫への耐性に合った昆虫食をレコメンド                    |
| 診断結果表示 | おすすめ昆虫の味・食感・難易度（★）とClaude Haiku 4.5が生成したパーソナライズコメントを表示 |
| 昆虫一覧   | 登録されている昆虫の一覧を閲覧できる                                     |
| 昆虫詳細   | 昆虫の詳細情報・レーダーチャート・Claude Haiku 4.5が生成した食レポコメントを表示       |

---

## 3. 診断ロジック

### 基本設計

| 項目      | 内容                                            |
|---------|-----------------------------------------------|
| テーマ     | 虫への耐性を測る                                      |
| 形式      | はい(1点) / いいえ(0点) の2択                          |
| 問題数     | 全6問（カテゴリ別にランダムピックアップ）                         |
| カテゴリ    | visual（視覚耐性）/ physical（身体耐性）/ mental（精神耐性）    |
| DB登録質問数 | visual: 4問 / physical: 4問 / mental: 4問（合計12問） |
| ピックアップ数 | visual: 2問 / physical: 2問 / mental: 2問（合計6問）  |
| スコア範囲   | visual: 0〜2点 / physical: 0〜2点 / mental: 0〜2点  |
| レコメンド   | 3軸スコアとDBの昆虫リストをClaude Haiku 4.5に渡してレコメンドを決定   |
| コメント生成  | レコメンドと同時にパーソナライズコメントを生成                       |
| 難易度表示   | ★1〜★5（昆虫データに設定した値を表示）                         |
| エラー処理   | APIエラー時はリトライを3回実施・それでも失敗の場合はデフォルトコメントを表示      |

### 質問カテゴリ

| カテゴリ     | 内容            | 問題例                  |
|----------|---------------|----------------------|
| visual   | 見た目・視覚的な耐性    | 虫が出てくる映像や画像を抵抗なく見られる |
| physical | 触る・食べる系の耐性    | 虫を手で触ったことがある         |
| mental   | 食への冒険心・精神的な耐性 | 食べ物の見た目より味や栄養を優先できる  |

### AIレコメンドの流れ

```
① Flutter がカテゴリ別スコアをGo/Ginに送る
② Go/Gin がDBから昆虫リストを全件取得
③ Go/Gin がスコアと昆虫リストをClaude Haiku 4.5に渡す
④ Claude Haiku 4.5 がリストの中から1つ昆虫を選びコメントを生成
⑤ Go/Gin が結果をFlutterに返す
```

> AIは必ずDBに登録済みの昆虫リストの中から選ぶため、未登録の昆虫がレコメンドされることはない
